# Session Startup

When beginning a new session on this codebase, always start by reviewing the architecture documentation:

1. Read `ARCHITECTURE.md` in the project root to understand:
   - Project structure and bundles
   - Key entities and data model
   - Technology stack
   - Common commands

2. Check for any recent changes or context from previous sessions

This ensures you have full context about the data-publisher application before making any changes.
